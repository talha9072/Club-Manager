== Description ==

Club Manager is a powerful yet easy-to-use WordPress plugin designed for clubs to efficiently manage users, memberships, events, and orders. Tailored specifically for clubs with multiple roles and needs, it provides a distraction-free experience while giving advanced users access to more complex functionality when needed.

Club Manager excels in the following areas:

* **Simplified User Management**: Easily track and manage members, event attendees, and customers with intuitive user dashboards. View, renew, or cancel memberships, manage user roles, and download membership eCards in just a few clicks.
  
* **Event and Product Management**: Create, manage, and track club events (free or paid), and sell memberships, tickets, or apparel from a dedicated dashboard. Club Managers have control over everything from orders to participant lists, with tools for easy exporting.

* **Customizable Experience**: Hide unnecessary features for a clean interface, or switch to an advanced view for more comprehensive options, allowing Club Managers to tailor their dashboard to their needs.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/club-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
